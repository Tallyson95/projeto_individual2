# projeto_individual2

Projeto desenvolvido no curso de Programação Web Programadores Cariocas.
O objetivo desse porojeto consiste em utilizados os conhecimentos de HTML, CSS e JavaScript para criar um codificador em base 64 e cifra de Cesar.
Para utilizar basta inserir uma mensagem que deseja criptografar, selecionar a opção de base 64 ou cifra de cesar.
Caso opte pela base 64, escreva a mensagem que deseja e confirme a criptografia.
Caso opte pela cifra de cesar, alem de escrever a mensagem desejada, é necessário tbm escolher o número de passos podendo esses passos serem positivos ou negativos.
